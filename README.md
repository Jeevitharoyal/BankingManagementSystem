# BankingManagementSystem
## Description 
The Bank Management System is a console-based Java application developed using Object-Oriented Programming concepts. It enebles users to perform basic banking operations such as creating accounts, depositing money,withdrawing money, checking account details, and managing customer information.

## Features
- Create Bank Account
- Display Account Details
- Search account
- Deposit Money
- Withdraw Money
- Update Account Details
- Delete account
- Save Data
- Load Data

  ## Technologies Used
  - Java
  - Eclipse IDE
  - Object-Oriented Programming(OOP)
  - ArrayList
  - File Handling
  - Serialization

    ## Project Structure
    - Main.java
    - bank.java
    - BankManager.java
    - FileManager.java

      ## How to Run
      1. Clone this repository.
      2. Open the project in Eclipse IDE.
      3. Run 'Main.java'.
      4. Use the menu to perform banking operations.
